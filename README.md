# WebCelSystem - Creator TOOL

<video src="https://github.com/user-attachments/assets/f9800902-5a93-479c-8aed-82da3d877348" width="100%" autoplay loop muted></video>

AnimeCelSystem 用アニメーションアセットを制作・管理するためのクリエイター向け統合開発環境です。

現在主流となっているLive2D制作ワークフローでは、

* モデル作成
* パラメータ設定
* モーション作成
* エクスポート

を個別ツールで管理することが一般的です。

本プロジェクトでは、

**「アニメーターがセル画感覚でVTuberを制作できる環境」**

を目指し、

* アセット管理
* パラメータ管理
* 中割り作成
* プレビュー
* ランタイム向けビルド

を一つのシステムへ統合しています。

---

# 開発背景

AnimeCelSystemでは、

リアルタイムにアニメーションセルを切り替えることで、

従来のVTuberモデルとは異なる

**アニメ的表現力**

を実現することを目標としています。

しかし、

アニメーション枚数が増加すると、

* ファイル管理
* パラメータ管理
* 中割り作業
* 出荷作業

の負担が急激に増加します。

本プロジェクトはその問題を解決するために開発されました。

アニメーターやイラストレーターが、

技術的な知識を最小限に抑えながら

AnimeCelSystem向けアセットを制作できる環境を提供します。

---

# システム概要

本システムは、

* ReactによるWebフロントエンド
* Goによるバックエンド

で構成されています。

クリエイターはブラウザ上で作業を行い、

最終的にAnimeCelSystemが利用する

* avatar.db
* .rgba

を含むZIPファイルとして出力できます。

# 技術概要

## フロントエンド

React + TypeScript により構築。

現在のパラメータ空間を視覚的に管理できるUIを実装しています。

---

## バックエンド

Go標準ライブラリを中心に構築。

* アセット管理
* RGBA変換
* ZIP生成
* SQLite生成

を担当します。

---

## データベース

制作中はMySQLを利用。

出荷時には、

演者用の軽量SQLiteデータベースへ変換します。

これにより、

* 編集効率
* 配布効率

を両立しています。

---

# 主要機能

## 3軸パラメータ編集

AnimeCelSystemが利用する

* Yaw
* Pitch
* Roll

をブラウザ上で編集できます。

現在位置に対応するアセットを即座に確認可能です。

---

## 進捗シグナル

各パラメータ点に対して、

* 未着手
* 作業中
* 完了

をリアルタイムに判定します。

状態は色として表示され、

制作状況を視覚的に把握できます。

---

## レイヤープレビュー

現在は

* Body
* Eyes
* Mouth

などのパーツを重ね合わせ表示できます。

実際のランタイムに近い状態を制作段階で確認できます。

---

## ドラッグ&ドロップアップロード

PNGをプレビューエリアへドロップするだけで、

現在選択中パラメータへ対応付けられます。

ファイル名は自動生成されます。

---

## SPLIT MODE

中割り制作支援機能です。

完成済みパラメータ間に新たな中間点を生成できます。

これにより、

作画枚数を段階的に増やしながら制作を進められます。

---

## SAVE as ZIP

制作したデータを、

AnimeCelSystem向けパッケージとして出力します。

生成内容：

* avatar.db
* .rgba assets

ZIP化された状態で即座にダウンロードできます。
